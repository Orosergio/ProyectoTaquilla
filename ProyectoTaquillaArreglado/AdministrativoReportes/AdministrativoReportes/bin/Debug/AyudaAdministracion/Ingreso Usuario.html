Formulario de Usuarios

Boton Ingresar

Para ingresar un Usuario debera seleccionar al empleado que se le asiganara el usario del campo Empleado, lugo seleccionar el Rol que este tendra, del campo Rol.

Debera escribir un nombre de usuario en el campo Nombre Usuario.

Luego debera escribir la contraseña y validarla con el campo Confirmar contraseña, al completar los campos debera presionar el boton de Ingresar para guardar los datos.
