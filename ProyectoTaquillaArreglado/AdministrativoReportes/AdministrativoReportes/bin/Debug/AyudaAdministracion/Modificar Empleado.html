Modificacion Empleado

En el campo Empleado se le mostraran los nombres de los empleados, ustede debera seleccionar el que desee modificar, al seleccionarlo, la tabla se llenara con los datos del empleado.

Para seleccionar los datos del empleado debera darle click al dato del empleado en la tabla, para que los datos se situen en la seccion de Datos anteriores.

En este punto debera llenar los nuevos datos del empleado, los cuales se llenaran el la seccion de Datos nuevos, al tener todos los campos completados, ya puede presionar el boton de modificar, para que los datos queden actualizados.
