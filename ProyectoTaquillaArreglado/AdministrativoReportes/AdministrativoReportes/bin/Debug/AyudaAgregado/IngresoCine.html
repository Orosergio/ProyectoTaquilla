Ingreso de Cines

Boton Ingresar

En este formulario ustede debera llenar los siguientes campos:

Nombre Cine: este campo llevara el nombre que se le dara al cine.

Tambien debera seleccionar el departamento y municipio en el que este cine estaran ubicados.

Direccion: En este campo debera escribir la direccion exacta del cine.

Tiempo: en este campo debera ingresar el tiempo que el cine le dara al cliente para poder reservar sus boletos o asientos.
