Modificacion de Usuario

Boton Modificar

En el campo Usuario se le mostraran los nombres de los Usuarios, ustede debera seleccionar el que desee modificar, al seleccionarlo, debera pesionar el boton de buscar para que los datos del usuario seleccionado se muestren en la tabla.

Para seleccionar los datos del Usuario debera darle click al dato del Usuario en la tabla, para que los datos se situen en la seccion de Datos anteriores.

En este punto debera llenar los nuevos datos del Usuario, los cuales se llenaran el la seccion de Datos nuevos, al tener todos los campos completados, ya puede presionar el boton de modificar, para que los datos queden actualizados.
