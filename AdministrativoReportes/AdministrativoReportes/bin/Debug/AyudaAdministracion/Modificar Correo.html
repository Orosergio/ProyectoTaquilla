Modificacion de Correos

Boton Modificar

En el campo correo debera escribir un correo y luego presionar el boton buscar, al presionar el boton de Buscar, le mostrara en la tabla los datos del propietario del correo, una vez los datos esten en la tabla debera selecconarlo para que se muestren en la seccion de Datos Anteiores, luego ya puede empezar a modificar los datos en la seccion de Datos nuevos, una vez esten todos los campos completos ya puede presionar el boton de midificar para que se guarden los datos.
